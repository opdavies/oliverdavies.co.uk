About Me

My name is Oliver Davies (aka opdavies) - a Web Developer and System Administrator based in Wales, UK. I’m a PHP specialist and Acquia certified Drupal Developer, a Senior Developer at Microserve, former Developer at the Drupal Association, part-time freelancer, open source contributor and advocate, mentor and conference speaker.

I’m a provisional member of the Drupal Security team and a webmaster and documentation maintainer for Drupal.org. I speak at various user groups and conferences, and co-organise the Drupal Bristol user group, PHPSW and DrupalCamp Bristol.
